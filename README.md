## 📆 프로젝트 개요
### 장애인분들을 대상으로 하는 일정 관리 웹서비스입니다.
### ✨ 주요 기능
- 장애인을 대상으로 하는 다양한 서비스 신청
- 달력을 통해 신청 내역과 개인 일정을 관리
- 자주 이용하는 서비스를 커스텀하여 음성인식으로 페이지 이동

## 🚀 실행 방법
사전 작업 : .env 파일 생성
```bash
REACT_APP_GOOGLE_MAP_API_KEY={본인 API키}
```

의존성 설치
```bash
npm install
```

실행
```bash
npm start
```
